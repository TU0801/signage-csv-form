# AskUserQuestionスキル

## 概要
ユーザーに選択肢を提示する際のAskUserQuestionツールの使い方。

## トリガー
- ユーザーに複数の選択肢を提示する時
- 作業の優先順位を確認する時
- 実装方法の選択肢を提示する時

## 改善ポイント

### やるべきこと
1. **選択肢を提示する時は必ずAskUserQuestionを使う**
   - テキストで選択肢を列挙するのではなく、チェックボックス形式で提示
   - multiSelect: true で複数選択可能に

2. **適切なオプション数**
   - 2-4個のオプション（多すぎると選びにくい）
   - 明確で簡潔なラベル（1-5単語）

3. **推奨オプション**
   - 推奨するオプションは先頭に配置
   - ラベルに「(推奨)」を追加

### やってはいけないこと
1. 選択肢をテキストで列挙するだけで終わらせる
2. 「どれにしますか？」と聞いてユーザーに入力させる
3. オプションが多すぎる（4個超え）

## 使用例

```javascript
{
  "questions": [{
    "question": "どの機能を優先しますか？",
    "header": "優先度",
    "options": [
      {"label": "認証機能", "description": "ログイン必須化"},
      {"label": "テスト追加", "description": "行詳細モーダルのテスト"},
      {"label": "CI/CD設定", "description": "GitHub Actionsの設定"}
    ],
    "multiSelect": true
  }]
}
```
