# 管理者画面改善スキル

## 概要
admin.htmlの現状と改善課題を記録。

## 現状の機能

### 実装済み
1. **統計ダッシュボード**
   - 総登録数
   - 今月の登録数
   - ユーザー数
   - 物件数

2. **データ一覧タブ**
   - フィルター（物件、期間）
   - 検索
   - 削除

3. **CSVエクスポートタブ**
   - フィルター
   - CSVダウンロード
   - クリップボードコピー

4. **マスター管理タブ**
   - 物件一覧表示
   - 受注先一覧表示
   - 点検種別一覧表示

5. **ユーザー管理タブ**
   - ユーザー一覧
   - 権限変更（user/admin）

## 未実装・改善課題

### 高優先度
1. **マスター追加機能**
   - 物件追加ボタン（#addPropertyBtn）のハンドラー未実装
   - 受注先追加ボタン（#addVendorBtn）のハンドラー未実装
   - 点検種別追加ボタン（#addInspectionBtn）のハンドラー未実装

2. **承認ワークフロー**
   - bulk.htmlからの申請を承認するフロー未実装
   - フッターに「申請する」ボタンがあるが、管理者側の承認機能がない

### 中優先度
3. **マスター編集・削除**
   - 各マスター項目の編集機能
   - 削除機能（使用中チェック付き）

4. **ユーザー追加・削除**
   - ユーザー招待機能
   - ユーザー削除機能

### 低優先度
5. **データエクスポート改善**
   - Excel形式でのエクスポート
   - フィルター条件の保存

## 対応ファイル
- admin.html
- js/admin.js
- js/supabase-client.js（API関数追加が必要）
