# 管理者画面改善スキル

## 現状の機能

### 実装済み
1. **統計ダッシュボード** - 総登録数、今月の登録数、ユーザー数、物件数
2. **承認待ちタブ** (v1.2.0) - 申請一覧、個別/一括承認・却下
3. **データ一覧タブ** - フィルター、検索、削除
4. **CSVエクスポートタブ** - フィルター、ダウンロード、コピー
5. **マスター管理タブ** (v1.2.0) - 物件・受注先・点検種別のCRUD
6. **ユーザー管理タブ** - ユーザー一覧、権限変更

## 未実装・改善課題

### 高優先度
1. マスター削除時の使用中チェック
2. データ一覧にステータス表示（pending/approved/rejected）

### 中優先度
3. ユーザー追加・削除機能
4. 却下時のフィードバック

### 低優先度
5. Excel形式エクスポート
6. 監査ログ
